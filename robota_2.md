# Звіт до роботи 2

## Тема
Основи програмування на Python

## Мета роботи
Навчитись застосовувати основні конструкції мови Python, виконати приклади з базових конструкцій та створити власні приклади з використанням AI.

## Виконання роботи

- Створено Python-файл для виконання базових прикладів
- Опрацьовано основні типи даних: `int`, `float`, `str`, `list`, `tuple`, `set`, `dict`
- Виведено вбудовані константи та зарезервовані слова Python
- Продемонстровано роботу вбудованих функцій (`abs`, `print`)
- Реалізовано приклади з циклами `for`
- Реалізовано приклади з розгалуженнями `if / else`
- Опрацьовано конструкцію `try / except / finally`
- Використано контекст-менеджер `with`
- Створено приклад з використанням `lambda`

### Приклад коду
```python
from datetime import datetime

name = "Назар"
city = "Львів"

print(f"Студент {name} з міста {city} почав вивчати Python у {datetime.now()}")
```
