# Звіт до роботи

## Тема
Вступні заняття: налаштування середовища, початок роботи з Python та Markdown

## Мета роботи
Налаштувати VS Code, створити GitHub-репозиторій, написати першу програму на Python та оформити звіт у Markdown.

---

## Виконання роботи

- Встановлено середовище розробки **Visual Studio Code**
- Інстальовано розширення **Python**, **Jupyter**, **GitHub Copilot**
- Створено GitHub-репозиторій та папку лабораторної роботи
- Створено файл `my_first_app.py` з першою програмою:

```python
from datetime import datetime

name = "Назар"
location = "Львів"

print(f"{name} start programming at {datetime.now()}. {location} is the best city!")
```

- Програму успішно виконано, виведено поточну дату та час
- Створено `my_first_app.ipynb` та додано Markdown-комірку з описом
- Усі файли закомічено та завантажено до GitHub

---

## Висновок

У ході роботи було налаштовано середовище розробки, створено GitHub-репозиторій та написано першу програму на Python. Мету роботи досягнуто, отримано базові навички роботи з Python, VS Code, GitHub і Markdown. Формат виконання роботи є зручним та зрозумілим.
